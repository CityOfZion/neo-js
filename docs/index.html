node

A class defining a node on the neo blockchain.

new node(options: Object)
Parameters
options (Object) Configuration parameters for the node.
Example
neo = require('neo-js-blockchain');
let options = {
  network: 'mainnet',
  storage: {
    model: 'mongoDB',
    collectionNames: {
      blocks: 'b_neo_m_blocks',
      transactions: 'b_neo_m_transactions',
      addresses: 'b_neo_m_addresses'
    }
  }
}
const n = node(options)
neo = require('neo-js-blockchain');
const n = node()
n.mesh.rpc('getBlock', 1000)

▸ getBalance(address, assets, blockAge)

Gets the balance of all assets and tokens for an address. This method will return the complete balance sheet for an account unless only a subset of assets is requested. The method also supports an optional blockAge attribute which will act as a caching mechanism to reduce compute load.

getBalance(address: String, assets: Array, blockAge: Number): any
Parameters
address (String) A contract address to get the balance of.
assets (Array = node.assets) An array of the assets to return balances for.
blockAge (Number = 1) getBalance uses a caching mechanic to reduce node load. If An asset's balance for an account has not been updated withing 'blockAge' blocks, it will retrieve an updated value. Increasing this number and substantial reduce computer load at the expense of balance discretization.
Returns
any: Promise. An array containing the balances of an address.
▸ getAssetBalance(address, asset, startBlock, balance)

Gets the balance of an asset belonging to a specific address on the blockchain. This method will also cache the result to the addresses collection.
